Projects

Club Archive is a a project I have been wokring on since November 2018, and it is a new way for clubs and organizations to gain an online presence. This project was implemented in Node.js, ExpressJS, and mongoDB and is deployed now at www.clubarchive.com! We also implmented a full CI/CD platform using Heroku for the deployment, and Travis CI for the server based testing. Check out the repo https://github.com/varvind/Club-Archive

Physics Equations Calculator is a project I worked on in 2016, and it was my first time developing something for the IOS Appstore! I wanted to come up with an app that made physics equations easier to solve, and I ended up getting 50 downloads among my peers! Application was written in swift.

HowdyHack was the first Hackathon I attended as a sophomore, and It was an awesome experience! I lead and developed the front end of a website we dubbed Maroon Pages, which was a .NET web app to help students find events on campus. I learned a lot about the google maps API, and C# web development! Check out the repo https://github.com/Jessepy/HowdyHackFall2019

Robomaster Robotics is a student led robotics organization that I worked for for about a year. I primarily worked on the computer vision team that used OpenCV to intelligently locate other robots on a field. Our robotics team competed in an international competition hosted by DJI in China in the summer of 2019, where we placed 3rd in the qualifying rounds. The software we were working on was as combination of C embedded code for the robot interfacing, and python for the openCV.

TAMUHack was the second hackathon I attended, and it was huge! I worked on a group of three, attended focused on the front end for a web app we built as a team, while also helping with the backend as well. The web app took user data from popular social media platforms such as Facebook, Instagram and Spotify and ran it through Microsoft Azure's sentiment analysis. This was primarily meant to address the issue of mental health among college students, and provide feedback based off of the data found on social media. The application was written in ExpressJS (nodeJS). Check out the repo https://github.com/varvind/TAMUHack-2020

Internship Experience

Role: Software Engineer Intern
From May 2020 - August 2020

My internship with Cirrus Logic was ROCKED! I essentially worked as a full time engineer and learned a lot in my time there. I primarily worked on one of the main internal and external software tools called Sound Clear Studio. This is a web based desktop Application written in Java (JavaFX framework) that interfaces with silicon that the companies produce. This is important for our customers to tune and use our devices to their specific needs.

I specifically worked on features and bug fixes for the project. Outside of coding itself, I worked on an agile based scrum team of 8 members, and attended daily standups, sprint reviews and backlog refinements (used Atlassian Jira and Confluence). The project also had a full CI/CD platform using Jenkins. Along with Sound Clear Studio, I worked on an internal python script that that used the Jira/Confluence REST API to query how many hours my coworkers logged, and emailed them if they did not reach a certain threshold. I also had to write a Jenkins pipeline script that schedules this script to run on certain days.
